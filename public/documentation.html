myFlix API

This API will provide users with information about different movies, directors, and genres. Users can sign up to create a list of their favorite movies.

Request documentation table below.

­­­­­Request

URL

HTTP Method

Request body data format

Response body data format

Get a list of ALL movies

/movies

GET

None

A JSON object holding data about all movies

Get data about a single movie by title

/movies/[Title]

GET

None

A JSON object holding data about a single movie containing a title, description, genre,  director, image URL, and whether it’s featured or not.

 

Example:

{

title: “Pulp Fiction”,

description: “American neo-noir black comedy crime film.”,

genre: { Crime, Drama},

director: “Quentin Tarantino”,

imageURL: “https://lh3.googleusercontent.com/proxy/bfon9rjsylO94vCIRhchuvKBMtCut2GGv0Urymj9aU6vCRRXx9Tg-FaZBYSikwKXCjKpg1j21WU6FUbV3P3fmz8vDSMS89MuBX9vGuwc_wdW9YOjPwMXR1eZRqLo7Opxdy0I6LXiGcA82UR6SWEqJgA” ,

featured: “No”

}

 

Get a description of genre by genre name

/genres/[Name]

GET

None

A JSON object holding genres and their respective descriptions

Get data about a single director by name

/directors/[Name]

GET

None

A JSON object holding data about a single director Example: { name: "Steven Speilberg", bio: "Steven Allan Spielberg is an American film director, producer, and screenwriter. He began his career in the New Hollywood era, and is currently the most commercially successful director.", birthYear: "1946", deathYear: "N/A", }

PUT

None

Allow a user to add movies to their list of favorites

/users/[username]/movielist

POST

None

A JSON object holding data about a user’s favorite movie titles: {

id: 5,

movielist: {movie title 1, movie title 2, movie title 3}

}

Remove a movie from a user’s list

/users/[username]/movielist

DELETE

None

A text message indicating a movie has been removed from a user’s list
