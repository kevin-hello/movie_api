myFlix API

This API will provide users with information about different movies, directors, and genres. Users can sign up to create a list of their favorite movies.

Request documentation table below.

Request URL HTTP Method Request body data format Response body data format
Get a list of ALL movies /movies GET None A JSON object holding data about all movies
Get data about a single movie by title /movies/[Title] GET None A JSON object holding data about a single movie
Example:
{
_id: "615bf0ddb0d8847c3857633a",
Title: "Pulp Fiction",
Genre: {
Name: "Crime"
Description: "Films of this genre generally involve various aspects of crime and its detection."
},
Director: {
Name: "Quentin Tarantino"
Bio: "Quentin Tarantino is an American film director, screenwriter, producer, author, film critic, and actor.",
Birth: "YYYY-MM-DD",
Death: "YYYY-MM-DD" (If Applicable, otherwise left blank)
}
Description: "Vincent Vega and Jules Winnfield are hitmen with a penchant for philosophical discussions. In this ultra-hip, multi-strand crime movie...",
ImagePath: "pulpfiction.png",
"Featured": true
}
Get a description of genre by genre name /genres/[Name] GET None A JSON object holding genre name and genre description.
Example:
{
Name: "Comedy",
Description: "Comedy is a genre of film in which the main emphasis is on humor. These films are designed to make the audience laugh through amusement and most often work by exaggerating characteristics for humorous effect."
}
Get data about a single director by name /directors/[Name] GET None A JSON object holding data about a single director.
Example:
{
Name: "Quentin Tarantino"
Bio: "Quentin Tarantino is an American film director, screenwriter, producer, author, film critic, and actor.",
Birth: "YYYY-MM-DD",
Death: "YYYY-MM-DD" (If Applicable, otherwise left blank)
}
Register a new user /users POST A JSON object holding data about a user.
Example:
{
"Username": "newuser808",
"Password": "newuserpass1",
"Email": "newuseremail@gmail.com",
"Birthday": "12/15/2000"
}
A JSON object holding data about a user.
Example:
{
_id: "615e37e0468a93cb35849356",
Username: "newuser808",
Password: "newuserpass1",
Email: "newuseremail@gmail.com",
Birthday: "2000-12-15T08:00:00.000Z",
FavoriteMovies: [],
}
Update a user's info by username /users/[Username] PUT A JSON object holding updated data about a user.
Example:
{
"Username": "newuser808",
"Password": "password$Movies",
"Email": "newuseremail@gmail.com",
"Birthday": "12/15/2000"
}
A JSON object holding updated data about a user.
Example:
{
_id: "615e37e0468a93cb35849356",
Username: "newuser808",
Password: "password$Movies",
Email: "newuseremail@gmail.com",
Birthday: "2000-12-15T08:00:00.000Z",
FavoriteMovies: [],
}
Add movies to a user's favorites list /users/[Username]/movies[MovieID] POST None A JSON object holding updated data about a user's favorites list.
Example:
{
_id: "615bff28b0d8847c3857637d",
Username: "movielover1",
Password: "123123",
Email:"movielover1@gmail.com",
Birthday: "1990-01-01T00:00:00.000Z",
FavoriteMovies: [ "615b58a30979c53e725c322c", "615bce82aa85b8228f70cf67", "615bd521aa85b8228f70cf69", "615bf0ddb0d8847c3857633a" ]
}
Remove a movie from a user's list /users/[Username]/movies[MovieID] DELETE None A JSON object holding updated data about a user's favorites list.
Example:
{
_id: "615bff28b0d8847c3857637d",
Username: "movielover1",
Password: "123123",
Email:"movielover1@gmail.com",
Birthday: "1990-01-01T00:00:00.000Z",
FavoriteMovies: [ "615b58a30979c53e725c322c", "615bce82aa85b8228f70cf67", "615bd521aa85b8228f70cf69" ]
}
Deregister a user /users/[Username] DELETE None A message indicating "[Username] has been deleted".
